
# Veekay Accessories

Veekay Accessories is an e-commerce app used to buy daily Accessories for men and women.




## Tech Stack

**Client:** `HTML`, `CSS`

**Hosting Platform:**   `Netlify`

- Product Listing Page

- Cart Management

- Authentication

    - Signup
